lib/auth.cljs

The file contains a simple system for authentication. It needs a lot of modification to actually be useful.

To be able to use it, you have to add the following to core/configure-server:

{% highlight clojure %} (.use app (.bodyParser express)) (.use app (.cookieParser express)) (.use app (.session express (util/map->js {:secret "secret!"}))) {% endhighlight %}

The "secret" is used to encrypt the contents of the session cookie (where username + password will be saved by).

You will also have to run (lib.auth/init-routes app), to enable login/logout capabilities.
