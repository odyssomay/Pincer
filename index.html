Pincer

Pincer is a kit for developing clojurescript sites. It contains a few scripts, some useful code, and some boilerplate.

Getting started

You can choose from two options, as you can see above.

The source version is the recommended to download. In this case you need the npm package manager for nodejs applications.

When you have pincer on your hard drive, start up your favorite terminal and run npm install.

To start working run sh scripts/main.sh. It is important that this is done from the pincer root directory.

If you haven't got any error messages, it's all good to go! Point your browser to http://localhost:8080/ and verify that everything is working properly.

Views

A view is an html file that you want to load from the browser. That is, in the same way that you would load a page from a static file server.

Views must be placed in the views directory. When in the views directory, they can be loaded in the browser with the path /page/:name, where :name is the name of the view. For example, if a file called test.html is placed in the views directory, it can be reached in the browser with /page/test.

Internals

Internals are exactly the same as views, but they are meant to be loaded from the current page. They are used to create a one page application. Note that you can mix views and internals however you like, meaning your application can mix the concepts of multiple page versus one page applications.

Partials

Partials are again, exactly the same as views, but they can only be imported from views or internals. This is used for parts that are shared between pages.
